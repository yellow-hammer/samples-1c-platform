# Участие

Репозиторий — **образцы с дефолтами**, а не полноценная конфигурация для прода.

- Новые объекты добавляйте **минимально заполненными** (как после создания в конфигураторе), без лишней бизнес-логики.
- Крупные или массовые правки по всему дереву — отдельное обсуждение / issue.
- Коммиты: [Conventional Commits](https://www.conventionalcommits.org/), сообщения **на русском** (по соглашению команды).
