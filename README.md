# 1c-platform-samples

Минимальные образцы артефактов **1С:Предприятие** с **значениями по умолчанию** платформы (как у только что созданного объекта). Это **ориентир по схеме выгрузки (Designer XML) и дефолтам новых объектов**, а не готовая прикладная конфигурация: копируйте, наполняйте, меняйте под свою задачу.

## Что внутри

| Каталог / тип                      | Назначение                                                     |
|------------------------------------|----------------------------------------------------------------|
| `src/cf/`                          | Выгрузка конфигурации в файлы (Designer XML).                  |
| `src/cfe/`, `src/epf/`, `src/erf/` | Выгрузки расширений конфигурации, внешних обработок и отчётов. |

## Snapshot-эталоны

В каталогах `snapshots/2.20/` и `snapshots/2.21/` лежат эталоны для автопроверок генерации:

- `cf/empty-configuration` — пустая выгрузка конфигуратора;
- `cf/empty-full-objects` — по одному объекту каждого поддержанного типа;
- `external-files/empty` и `external-files/empty-full-objects` — минимальные внешние артефакты.

Эти файлы используются в тестах `md-sparrow` как контрольные образцы выходного XML.
